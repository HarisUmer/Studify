## ✨ Overview

Studify uses **CameraX** to stream the device camera and recognize **hand gestures** frame-by-frame, turning them into app actions. It's a hands-on exploration of Android's modern camera stack combined with on-device gesture/vision inference.

## 🧠 Highlights

- 📷 **CameraX pipeline** — modern, lifecycle-aware camera capture and analysis.
- ✋ **Real-time gesture recognition** on the live camera feed.
- 📱 Built for Android.

## 🚀 Build

Open the project in **Android Studio**, let Gradle sync, and run on a device/emulator with a camera.
